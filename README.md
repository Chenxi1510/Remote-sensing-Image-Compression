# Remote-sensing-Image-Compression
Remote-sensing Image Compression


This repository contains the code for reproducing the results with trained models (EGA-Net), NWPU-RSC Dataset, and a novel full-reference IQA FITS.

## Requirements

Python==3.6  
torch==1.6.0  
torchvision==0.7.0
